### JLaTexMath for Android
完美支持、数学、物理、化学公式、汉字解析、图文混排

本项目借鉴自[jlatexmath-android](https://github.com/mksmbrtsh/jlatexmath-android)（由java Api 转化来的Android版本）

本项目针对化学、生物类的中文公式做了优化。
